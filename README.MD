# Armenian Spell Check for VS Code
## Հայերեն ուղղագրության ստուգում VS Code-ի համար



This is an Armenian dictionary extension for the [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker). It helps identify and correct Armenian spelling errors directly within your code, comments, and strings.

Սա հայերեն բառարան-հավելում է [Code Spell Checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker) հավելվածի համար։ Այն օգնում է գտնել և ուղղել հայերեն ուղղագրական սխալները անմիջապես կոդի, մեկնաբանությունների և տեքստերի մեջ։

---

## Features / Հնարավորություններ

- **Comprehensive Dictionary:** Includes thousands of Armenian words with their inflections.
- **Հարուստ բառարան:** Ներառում է հազարավոր հայերեն բառեր՝ իրենց հոլովված և խոնարհված ձևերով։
- **Fast Performance:** Uses an optimized `Trie` format for instant feedback.
- **Արագագործություն:** Օգտագործում է օպտիմալացված `Trie` ձևաչափ՝ ակնթարթային աշխատանքի համար։
- **Case Sensitivity:** Correcty handles Armenian capital and lowercase letters.
- **Մեծատառ/Փոքրատառ:** Ճիշտ է աշխատում հայերենի մեծատառերի և փոքրատառերի հետ։

## Installation / Տեղադրում

### Option 1: VSIX (Manual)
1. Download the `.vsix` file from the [Releases](https://github.com/dhayk/ArmenianSpell_Check/releases) section.
2. Open VS Code and go to the Extensions view (`Ctrl+Shift+X`).
3. Click on the `...` (top right) and select **Install from VSIX...**.

### Տարբերակ 1: VSIX (Ձեռքով)
1. Ներբեռնեք `.vsix` ֆայլը [Releases](https://github.com/dhayk/ArmenianSpell_Check/releases) բաժնից։
2. Բացեք VS Code-ը և գնացեք Extensions բաժին (`Ctrl+Shift+X`):
